# aaronrdavis.news

Personal portfolio site for Aaron R. Davis — investigative journalist.

## Deployment

Managed via Wrangler CLI on Watts (Tower). GitHub is canonical source.

```bash
wrangler pages deploy . --project-name=aaronrdavis-news
```

## Structure

- `index.html` - Landing / credentials
- `about.html` - Professional background
- `contact.html` - Contact information
- `style.css` - Shared styles

## Domain

Hosted on Cloudflare Pages with custom domain: **aaronrdavis.news**
